# Diseno visual

## Paleta

La paleta base viene de la referencia visual proporcionada:

| Uso | Color |
| --- | --- |
| Fondo suave | `#F3CCDE` |
| Superficies secundarias | `#D6A8C4` |
| Acento medio | `#BA88AE` |
| Acento fuerte | `#9E6899` |
| Texto principal / sidebar | `#5B3765` |

## Estilo esperado

La app debe sentirse academica, ordenada y cuidada. No debe parecer una landing
page ni una coleccion de CRUDs.

## Principios de UI

- Dashboard operativo como primera pantalla despues del login.
- Sidebar sobrio y util.
- Tarjetas con metricas reales, no solo numeros decorativos.
- Formularios con labels, errores visibles y estados de carga.
- Tablas o listas escaneables para tareas, materias y recursos.
- Colores suaves para fondos y morado profundo para acciones importantes.
- Evitar emojis como base principal del diseno; usar iconos o texto claro.
- Responsive desde el inicio.
