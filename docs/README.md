# Documentacion

Esta carpeta queda separada en dos partes para que no se mezclen documentos de
apoyo con entregables finales.

## entregables

Archivos finales para entregar en la materia:

- plan de pruebas en PDF.
- reporte de bugs.
- archivo `.jmx` de JMeter.
- reporte generado por OWASP ZAP.
- evidencias manuales si se necesitan.

Las pruebas unitarias y Cypress no se copian aqui como entregable porque ya
estan dentro del codigo del proyecto.

## proyecto

Documentos de apoyo para entender el proyecto:

- estructura de carpetas.
- guia para ejecutar backend, frontend y pruebas.
- ruta de trabajo.
