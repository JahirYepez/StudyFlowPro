# StudyFlow

Proyecto full-stack para la materia Calidad en el Desarrollo de Software.

StudyFlow sera una aplicacion academica para organizar materias, tareas, metas,
sesiones de estudio, recursos y repasos, con enfoque en calidad: pruebas,
logging, seguridad basica, documentacion y despliegue.

## Stack decidido

- Frontend: React + TypeScript + Vite
- Backend: FastAPI
- Base relacional: PostgreSQL
- Base no relacional: MongoDB
- Pruebas: pytest, Cypress, JMeter y OWASP ZAP
- Observabilidad: logging estructurado JSON con correlation ID
